# Product Inventory API

A simple API for managing a product inventory built using Django REST Framework.

## Features

- List all products
- Add a new product
- Update a product
- Delete a product
- **Bonus:** Validation to ensure the product price is non-negative

---

## Requirements

- Python 3.9+
- Django 4.x
- Django REST Framework 3.x
